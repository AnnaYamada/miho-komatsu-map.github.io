小松未歩マップ
====

[小松未歩マップ](https://miho-komatsu-map.github.io/)は、小松未歩さんのプロモーションビデオ撮影場所やその他の地図情報をまとめたGoogleマイマップと、関連情報です。このマイマップをあなたのGoogleマップにマイプレイスとして設定すれば、未歩さんにまつわる場所を簡単にめぐることが出来ます。ぜひご活用ください！
